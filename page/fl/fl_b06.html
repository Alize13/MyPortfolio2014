圖片捲動

圖片自動捲動，使用者也可以藉著滑鼠控制圖片捲動

抓取滑鼠位置決定往左或往右移動，正中央則靜止

滑鼠越靠近邊邊移動的越快，用靠近中間越慢

 

作品展示

程式碼

addEventListener(Event.ENTER_FRAME, $move);

function $move($e:Event){
	pic.x-=(mouseX-stage.stageWidth/2)*0.02;
	/*重複一直移動的方式
	if(pic.x>1800){
		pic.x-=pic.width/2;}
	if(pic.x< -1200){
		pic.x+=pic.width/2;}*/
	
	//移動到最邊邊就停住的方式
	if(pic.x>1800){
		pic.x=1800;
	}
	if(pic.x<-1200){
		pic.x=-1200;
	}
}
